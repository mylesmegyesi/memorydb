# Repository

An abstraction for using the reporitory pattern with many different backends.

## Usage

### Mongo with PORO models

``` ruby
gem 'repository', '0.0.1'
gem 'mongo',      '1.8.1'
gem 'bson_ext',   '1.8.1'
```

``` ruby
require 'repository/mongo'
```

### In-memory with PORO models

``` ruby
gem 'repository', '0.0.1'
```

``` ruby
require 'repository/memory'
```
